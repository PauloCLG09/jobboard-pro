# 💼 JobBoard Pro

Aplicación web moderna de bolsa de trabajo construida con **Next.js, TailwindCSS y React Context API**.
Permite explorar ofertas laborales, guardar favoritas y disfrutar de una experiencia de usuario fluida con animaciones.

---

## 🚀 Demo en vivo

👉 https://TU-LINK-VERCEL.vercel.app

---

## ✨ Funcionalidades

* 🔍 Explorar ofertas de trabajo
* 💾 Guardar y eliminar empleos (persistencia con LocalStorage)
* 🔔 Notificaciones tipo toast (feedback en tiempo real)
* 🎨 Interfaz moderna con TailwindCSS
* 🎬 Animaciones suaves con Framer Motion
* 📱 Diseño responsive
* 📂 Estado vacío (empty state) para mejor UX

---

## 🧠 Tecnologías utilizadas

* Next.js (App Router)
* React
* TypeScript
* TailwindCSS
* Framer Motion
* Context API + LocalStorage
